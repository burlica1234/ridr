# Documentation

## 🧩 Project Overview

Ridr is a micro-mobility platform designed for:
- bicycles
- e-scooters
- skateboards
- other lightweight urban transport

The platform provides:
- smart routing
- parking discovery
- legal awareness
- community reporting

---

| Area | Technology |
|------|-----------|

| Security | Spring Security + JWT |
| ORM | Spring Data JPA |
| Mapping | MapStruct |
| Validation | Jakarta Validation |
